FONTS DIRECTORY
===============

Required font: DejaVuSans.ttf
Purpose: Unicode font enabling Hindi (Devanagari) text in PDF reports.

HOW TO DOWNLOAD:
1. Visit: https://dejavu-fonts.github.io/
2. Download the DejaVu fonts zip
3. Extract DejaVuSans.ttf from the zip
4. Place it in this /fonts/ directory

OR use this direct link:
https://github.com/dejavu-fonts/dejavu-fonts/releases/download/version_2_37/dejavu-fonts-ttf-2.37.tar.bz2

The app will gracefully fall back to English-only PDF if this file is missing.
Hindi PDF section will be skipped with a note explaining how to enable it.
